# Projects — JavaScript Fundamentals

## How to use

- Each topic folder `NN-.../` maps to one example lesson (`examples/NN-...md`).
- Each exercise is `NNN-.../` with `README.md`, `starter/`, and optional `solution/`.
- Read the lesson first, then the project README, then code in `starter/`.

## How to run

From a project folder:

```bash
node starter/index.js
```

Use Node LTS. Confirm with `node -v`.

## What you should practice

- Applying each lesson’s stack-first mental model in runnable code
- Explicit validation and clear error messages
- Small functions, predictable I/O, acceptance criteria from each project README

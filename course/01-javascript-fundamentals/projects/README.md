# Projects — JavaScript Fundamentals

## How to use
- Each topic lives in a folder `NN-.../`.
- Each exercise lives in a folder `NNN-.../` with a `README.md`, a `starter/` folder, and (optionally) a `solution/` folder.

## Goal of this track
- Practice writing **explicit** checks instead of relying on coercion.
- Learn to validate and normalize CLI inputs safely (common in HackerRank).

# Projects — Promises

Practice exercises for this lesson. Run each project with:

```bash
node starter/index.js
```

## What you should practice

- `new Promise` executor with resolve/reject
- `.then` chaining and `.catch` for errors
- Wrapping callback APIs in Promises

## Project catalog

### 002-retry-with-backoff/
